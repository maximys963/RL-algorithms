## Hello my friend this little readme file help you set up this hell laba

1. Clone this repo
```
git clone https://github.com/maximys963/RL-algorithms.git

```
2. Open folder laba1.4
3. Install node packages 
```
yarn install

```
or 
```
npm install
```
4. run project

```
yarn start 

```
or
```
npm run start

```
